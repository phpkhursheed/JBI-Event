INTRODUCTION
------------

JBI Event module is developed for creating custom content type name JBI Event and custom taxonomy Event Category which stores the events data in backend and display the Events list by Category with Subscribe Options.

Simply put the modules under modules/custom directory and just enabled the modules.

- You will get an option to create JBI Event from Admin Content.
- Added data will display on the frontend on below url.
- /jbi_event/data
	Example - http://abc.com/jbi_event/data


TO ACCESS JBI EVENTS SUBSCRIBER LIST
------------------------------------

- /admin/content/jbi_event/subscriber
   Example - http://abc.com/admin/content/jbi_event/subscriber
